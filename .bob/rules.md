# Bob's Development Rules

## Planning & Approval
- Always write implementation plan to phases.md before writing any code
- Never start coding without my explicit approval
- Split all implementations into numbered phases

## Technology Stack
- Use FastAPI for backend
- Use React functional components for frontend
- Use SQLite database

## Security & Configuration
- Never hardcode API keys, use environment variables
- Add .env to .gitignore

## Session Management
- Export Bob sessions after each phase completes
