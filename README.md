# EduBob

AI-powered development and learning loop using IBM Bob.

## Problem

Educators spend hours creating assignments. Students get feedback too late. Developers struggle to understand codebases.

## Solution

EduBob uses IBM Bob to generate assignments, review code, and explain codebases.

## Features

* Assignment Generator
* Code Review
* Codebase Analyzer
* Validation Engine
* Mistake Pattern Memory

## Tech Stack

* FastAPI
* React
* SQLite
* IBM Bob
